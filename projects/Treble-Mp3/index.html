    Treble Mp3

    Music app that allows users to create a library from youtube, spotify, or manually by adding an MP3 File with a thumbnail.

    Listeners can share the music on different platforms.

    This project interests me because

    • I love music based application
    • Allows listners to download music from youtube, without the premium membership

    Skill set Showcase: - It shows that I can create a Full Stack Application with Authentication, Frontend in NextJS, Backend Server components, and AWS s3 for storing images and mp3 files.
